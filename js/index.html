The IBM System/360 Model 50 was a popular mainframe introduced in 1964. This web page simulates the mainframe by running the original microcode. The console lights are illuminated as they would be on an real Model 50.
To start the simulator, click on the Run button below. The console lights are controlled by rollers; click on the roller knob to change the display function. You can zoom in on the console display for more detail.
